### Prerequisites
This project was built using:

| Program       | Version       |
| ------------- |:-------------:|
| node          | 10.1.0        |
| npm           | 5.8.0         |
| angular cli   | 6.0.0         |

### Get started 
Install all node packages:
```
npm i
```

To run in development mode:
```
npm run start:dev
# or #
ng serve
```

To run using angular universal and a node server use:
```
npm start
```
